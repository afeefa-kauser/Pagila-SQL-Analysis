# Pagila SQL Analysis Project

## Project Overview

This project demonstrates SQL data analysis skills using the Pagila sample database in PostgreSQL. The project focuses on extracting meaningful insights through SQL queries and performing analysis on customers, payments, rentals, and sales data.

## Objectives

* Practice PostgreSQL concepts
* Perform real-world data analysis
* Improve query writing skills
* Build portfolio projects for SQL and Data Analyst roles

## Database Used

Pagila Sample Database

## SQL Concepts Used

* SELECT statements
* WHERE conditions
* GROUP BY
* ORDER BY
* Aggregate functions
* JOIN operations
* Subqueries
* CASE statements
* Date functions

## Analysis Performed

### Payment Analysis

* Total payments
* Revenue calculations
* Daily sales analysis
* Payment range filtering

### Customer Analysis

* Customer activity analysis
* Rental behavior
* Customer payment trends

### Sales Analysis

* Highest sales amount
* Top performing employees
* Daily sales comparison

## Technologies Used

* PostgreSQL
* SQL
* pgAdmin

## Outcome

This project helped strengthen SQL querying and analytical thinking skills through practical database analysis tasks.
